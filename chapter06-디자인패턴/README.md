# 디자인 패턴
> 소프트웨어 공학에서 `디자인 패턴(Design pattern)`은 프로그램 개발 시에 자주 부닥치는 애로 상황에 대한 일반적이고 재사용 가능한 추상화된 해결책을 일컫는 말입니다.

이번 장에서는 각 언어별 디자인 패턴을 알아봅니다. 깊진 않고 간략하게.
