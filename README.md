# Developer's Note
### 개요
:star2:리뉴얼중입니다!:star2:

### 마크다운 작성 규칙
> 중요하지 않아요!:blush:

1. `*.md` 파일의 첫 헤드라인은 `#`을 사용합니다.
2. 항목별 정리 시 `##`를 사용하고, 소항목은 `###`입니다.
  - `##` 항목의 끝에 상위 문서로 갈 수 있는 링크를 붙입니다.
    ```
    ## 예시
    - 내용

    <br>

    <sup>[(상위 문서로)](https://github.com/InSeong-So/IT-Note)</sup>

    <hr>
    <br>

    ## 예시
    - 계속...
    ```
3. TOC는 생성하지 않습니다.
4. 문서 내의 한 항목에서 설명하는 내용이 100줄을 넘어가면 분리합니다.
5. 바로가기 링크는 `Link`로 표현합니다.
